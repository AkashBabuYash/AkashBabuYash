## 🤖 Featured Project: AI Agent PPT Generator

### 📌 Overview
AI-powered system that generates PowerPoint presentations from a given topic using LLMs.

### ⚙️ Workflow
1. User inputs topic  
2. AI generates structured content  
3. Slides are organized automatically  
4. PPT is created using Python  

### 🧩 Architecture
- Planner Agent → decides structure  
- Content Agent → generates text  
- Formatter Agent → formats slides  
- Tool Agent → generates PPT  

### 🔧 Tech Used
- LangChain  
- LangGraph  
- LangSmith  
- Python (python-pptx)  
- FastAPI  

### 🚀 Features
- Auto PPT generation  
- Structured slides  
- Saves time
